# TableFooter
Container for summary rows or pagination beneath table body.

## Import
```tsx
import { TableFooter } from '@xanui/ui';
```

## Usage
```tsx
<TableFooter>
  <TableRow>...</TableRow>
</TableFooter>
```

## Props
Inherits Tag props.

## Best Practices
Use for aggregate values or controls (pagination). Keep design lighter than header.
