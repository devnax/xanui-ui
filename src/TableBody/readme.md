# TableBody
Wrapper for table rows inside `Table`.

## Import
```tsx
import { TableBody } from '@xanui/ui';
```

## Usage
```tsx
<TableBody>
  <TableRow>...</TableRow>
</TableBody>
```

## Props
Inherits Tag props.

## Best Practices
Keep row content consistent with header columns; avoid mixing interactive and non-interactive rows without clear affordances.
