# TableHead
Holds column header cells.

## Import
```tsx
import { TableHead } from '@xanui/ui';
```

## Usage
```tsx
<TableHead>
  <TableRow>
    <TableCell>ID</TableCell>
  </TableRow>
</TableHead>
```

## Props
Inherits Tag props.

## Best Practices
Use concise labels; include sort indicators if interactive.
