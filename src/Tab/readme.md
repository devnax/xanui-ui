# Tab
Individual tab header item used within Tabs container.

## Import
```tsx
import { Tab } from '@xanui/ui';
```

## Usage
```tsx
<Tab value="profile">Profile</Tab>
```

## Props (summary)
| Name     | Type      | Description            |
| -------- | --------- | ---------------------- |
| value    | string    | Tab identifier.        |
| selected | boolean   | Active state.          |
| disabled | boolean   | Non-interactive state. |
| children | ReactNode | Label content.         |

## Best Practices
Ensure clear contrast for selected state; keep labels short.
